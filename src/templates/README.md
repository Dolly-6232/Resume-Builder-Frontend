# Resume Templates

This directory contains the template definitions for various resume styles.

## Available Templates

1. **Modern** - Contemporary design with modern layout
2. **Fresher** - Ideal for fresh graduates
3. **Simple** - Clean and minimal design
4. **Corporate** - Professional corporate style
5. **Creative** - Creative and artistic layouts
6. **Startup** - Startup-friendly design
7. **Executive** - Executive-level resume style
8. **Professional** - Professional classic style
9. **International** - International resume format
10. **ATSPro** - Optimized for ATS compatibility

Each template directory contains the template configuration and styling details.
